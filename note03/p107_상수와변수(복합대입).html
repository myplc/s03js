상수와 변수, 복합대입연산자

  • const 는 변한지 않는 값을 선언할 때 : const 이름 = 값;

  • let 은 변할 가능성이 있는 값을 선언할 때 : let 이름 = 값;

  • 복합대입연산자는 대입연산자와 다른연산자를 함께 사용하는 연산자

    • 복합대입연산자 설명 사용예 의미
    += 기존 변수의 값에 값을 더합니다. a+=1 a=a+1
    -= 기존 변수의 값에 값을 뺍니다. a-=1 a=a-1
    *= 기존 변수의 값에 값을 곱합니다. a*=1 a=a*1
    /= 기존 변수의 값에 값을 나눕니다. a/=1 a=a/1
    %= 변수의 값의 나머지를 구합니다. a%=1 a=a%1
